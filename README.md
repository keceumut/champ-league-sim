# Champions League Simulator

A 4-team football league simulation built with **Laravel 11** and **Vue 3**.

## What it does

- Configure 4 teams with custom names and strength ratings (60–95)
- Simulate a round-robin season week by week or all at once (6 weeks, 12 matches)
- Live league table with P / W / D / L / GF / GA / GD / Pts
- Edit any past match result and watch standings update instantly
- Championship odds prediction based on points-per-game projection (unlocks after week 4)
- **Bulk simulation mode** — run the full season N times (10–100) and see statistical win rates per team

## Tech stack

| Layer    | Technology                        |
|----------|-----------------------------------|
| Backend  | PHP 8.2 / Laravel 11              |
| Database | SQLite (file-based, no server)    |
| Frontend | Vue 3 + Vue Router + Vite         |
| Styling  | Tailwind CSS                      |
| Tests    | PHPUnit                           |

## Local setup

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate

npm install
npm run dev        # Vite dev server (hot reload)
php artisan serve  # Laravel at http://localhost:8000
```

Open **http://localhost:8000** — Vite hot-reload wires in automatically.

## Running tests

```bash
php artisan test
```

Tests use an in-memory SQLite database and cover:

- `LeagueServiceTest` — standings calculation (wins, draws, sorting by GD)
- `PredictionServiceTest` — projected-points formula (PPG × 6)
- `BulkSimulationServiceTest` — bulk run structure, win-count totals, sorting
- `SimulationControllerTest` — full API endpoint coverage (setup → simulate → reset → edit)

## Simulation model

Goals per team are generated by giving each team 10 scoring attempts; the probability per attempt scales with the team's strength (`strength / 100 × 0.3`). A strength-90 team scores ~2.7 goals on average; a strength-60 team scores ~1.8. This produces realistic-looking scorelines while still allowing upsets.

Championship odds are computed as each team's share of the total projected points among teams that can still mathematically reach the current leader.
